# Qwen3-ASR 语音识别模型 API 文档

通义千问系列的录音文件识别模型能将录制好的音频转换为文本，支持多语言识别、歌唱识别、噪声拒识等功能。

## 核心功能

- **多语种识别：** 支持多语种语音识别（涵盖普通话及多种方言，如粤语、四川话等）。
- **复杂环境适应：** 具备应对复杂声学环境的能力，支持自动语种检测与智能非人声过滤。
- **歌唱识别：** 即使在伴随背景音乐（BGM）的情况下，也能实现整首歌曲的转写。
- **上下文增强：** 通过配置上下文提高识别准确率。
- **情感识别：** 支持多种情绪状态识别（涵盖惊讶、平静、愉快、悲伤、厌恶、愤怒和恐惧）。

## 适用范围与模型

服务主要提供两大核心模型，分别适用于不同的场景：

### 通义千问3-ASR-Flash (Sync)
- **模型名称**: `qwen3-asr-flash`
- **用途**: 专为短音频（最长5分钟）的同步或流式识别设计。
- **场景**: 语音消息、实时字幕等。
- **特点**: 低延迟，支持流式返回。

### 通义千问3-ASR-Flash-Filetrans (Async)
- **模型名称**: `qwen3-asr-flash-filetrans`
- **用途**: 专为长音频（最长12小时）的异步识别设计。
- **场景**: 会议记录、访谈整理等。
- **特点**: 支持长音频，具备情感识别与句粒度时间戳功能，适合后期索引与分析。

## 模型选型指南

| 场景 | 推荐模型 | 理由 | 注意事项 |
| :--- | :--- | :--- | :--- |
| **长音频识别** | `qwen3-asr-flash-filetrans` | 支持最长12小时录音，具备情感识别与句粒度时间戳功能 | 音频文件大小不超过2GB，时长不超过12小时 |
| **短音频识别** | `qwen3-asr-flash` | 短音频识别，低延迟 | 音频文件大小不超过10MB，时长不超过5分钟 |
| **客服质检** | `qwen3-asr-flash-filetrans`<br>`qwen3-asr-flash` | 可分析客户情绪 | 根据音频时长选择合适的模型 |
| **字幕生成** | `qwen3-asr-flash-filetrans` | 长音频+标点预测+时间戳，直接生成结构化字幕 | 需后处理生成标准字幕文件 |
| **多语种视频本地化** | 两个模型均支持 | 覆盖多种语言+方言 | 根据音频时长选择合适的模型 |
| **歌唱类音频分析** | 两个模型均支持 | 识别歌词并分析情绪 | 根据音频时长选择合适的模型 |

更多详细信息请参考具体 API 文档页面。
